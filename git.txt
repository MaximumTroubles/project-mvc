Глобальные данные 1 раз в самом начале 
git config --global user.name "Maxym Rodziyevsky"
git config --global user.email "Rodziyevsky@gmail.com"


Инициализация git
git init

Статус 
git status

Добавить новый файлы  (все)
git add . 


Коммит (фиксация изменений)
git commit -m "First commit"


Логи (все комииты )
git log 
git log --pretty=oneline

Просмтореть какие изменения были в  комите
git show (id commita)

Добавление ветки
git branch dz
git checkout dz - переход в ветку


git checkout -b dz - создание и переход в новую ветку
