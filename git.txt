Глобальные данные 1 раз в самом начале 
git config --global user.name "Maxym"
git config --global user.email "Rodziyevsky2@gmail.com"


Инициализация git
git init

Статус 
git status

Добавить новый файлы  (все)
git add . 


Коммит (фиксация изменений)
git commit -m "First commit"


Логи (все комииты )
git log 
git log --pretty=oneline

Просмтореть какие изменения были в  комите
git show (id commita)

Добавление ветки
git branch dz
git checkout dz - переход в ветку


git checkout -b dz - создание и переход в новую ветку


Обьединить ветки 
git merge dz (при этом мы были в ветке master)

Удалить ветку 
git branch -D dz


create a new repository on the command line


echo "# project-mvc" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:MaximumTroubles/project-mvc.git
git push -u origin main
